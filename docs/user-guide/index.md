# CONFORMITAS 3.0 — Manual do Usuário

**Bem-vindo ao CONFORMITAS 3.0**, o Sistema Integrado de Gestão de Auditoria Interna do Tribunal de Justiça do Estado do Ceará.

## O que é o CONFORMITAS?

O CONFORMITAS é a plataforma oficial da **Secretaria de Auditoria Interna (AUDIN)** do TJCE. Ele organiza todo o ciclo de auditoria interna — do planejamento anual ao acompanhamento das recomendações — em um único sistema digital, substituindo planilhas e controles manuais.
