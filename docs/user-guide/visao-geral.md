# Visão Geral — CONFORMITAS 3.0

## O que o CONFORMITAS faz?

O CONFORMITAS é o sistema que organiza digitalmente o ciclo completo de auditoria interna do TJCE. Ele substitui as planilhas, documentos soltos e controles manuais por uma plataforma integrada onde tudo fica registrado, rastreável e acessível conforme o perfil de cada usuário.

## Os 4 grandes blocos do sistema

### 1. Planejamento
A AUDIN define **o que vai auditar** e **quando**. O sistema ajuda a priorizar as áreas mais críticas usando uma matriz de risco, planejar as horas da equipe e submeter o Plano Anual de Auditoria (PAA) para aprovação da Presidência.

### 2. Execução
Os auditores executam o trabalho de campo: emitem comunicados às unidades, coletam documentos (evidências), registram verificações (papéis de trabalho) e identificam fatos relevantes (achados) — tudo dentro do sistema.

### 3. Resultados e Recomendações
Os achados são enviados às unidades auditadas para manifestação. Após a resposta (ou expiração do prazo), a AUDIN emite o Relatório Final com recomendações. Cada recomendação tem um prazo e um responsável.

### 4. Monitoramento e Qualidade
O sistema acompanha se as recomendações estão sendo cumpridas, alerta sobre prazos vencidos e permite que a própria AUDIN avalie a qualidade do seu trabalho (PQAUD).

## Fluxo principal

```
Planejamento (PAA) → Abrir auditoria → Coletar evidências →
Registrar achados → Unidade se manifesta → Relatório Final →
Recomendações → Monitoramento → Qualidade
```

## Módulos do sistema

| Módulo | O que faz |
|--------|-----------|
| **Planejamento** | Cadastra áreas auditáveis, gera matriz de priorização, cria PALP e PAA |
| **Execução** | Abre auditorias, gerencia evidências e papéis de trabalho |
| **Achados** | Registra fatos relevantes com 4 atributos (situação, critério, causa, efeito) |
| **Relatórios** | Gera Relatório Preliminar, Final e Relatório Anual |
| **Monitoramento** | Acompanha recomendações até implementação |
| **Consultorias** | Registra pedidos de assessoramento de outras unidades |
| **Qualidade** | Gerencia autoavaliações e avaliações externas (PQAUD) |
| **Dashboards** | Painéis visuais com indicadores de desempenho |
